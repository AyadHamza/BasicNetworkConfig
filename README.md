<h1>Basic Network Configuration<h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
This Project consists of one router and two switches, as well as four endhosts/workstations. There will be two differnet network addressing scheme/networks. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Cisco Packet Tracer</b> 

<h2>Environments Used </h2>

- <b>Cisco Packet Tracer</b>
- <b>Cisco IOS</b>
- <b>Windows 11<b>
